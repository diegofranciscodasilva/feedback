# Página de Feedback

Bem-vindo à Página de Feedback! Este projeto é uma página web desenvolvida com HTML, CSS e JavaScript, oferecendo aos usuários a opção de fornecer feedback por meio de escolhas, envio e exibição da resposta.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript

## Como Visualizar

Baixe ou clone o repositório em seu computador.
Abra o arquivo `index.html` em seu navegador web.

## Recursos

- **Escolha de Opção**: Os usuários podem selecionar uma opção de feedback.
- **Formulário de Envio**: Um formulário simples permite que os usuários enviem seu feedback.
- **Exibição da Resposta**: Após o envio, a resposta é exibida na página.
- **Design Intuitivo**: Interface projetada para simplicidade e facilidade de uso.

## Estrutura do Projeto

- `index.html`: Contém a estrutura HTML da página principal.
- `styles.css`: Arquivo de estilo CSS para estilizar a página.
- `script.js`: Script JavaScript responsável pela lógica interativa.

## Personalização

- Modifique o arquivo `index.html` para ajustar a estrutura conforme necessário.
- Adapte o estilo no arquivo `styles.css` para atender às suas preferências.
- Explore o arquivo `script.js` para personalizar a lógica interativa ou adicionar mais funcionalidades.

## Pré-requisitos

Certifique-se de ter um navegador web instalado para visualizar o projeto.
